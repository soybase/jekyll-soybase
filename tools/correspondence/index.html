Gene Model Correspondence Lookup

There are several good options for identifying corresponding genes in different accessions or annotations. If you have ...
  • One or a few genes to look up? Use the pangene search tool.
  • Many genes to look up among reference** accessions? Download a correspondence table for the reference lines .
        **currently: all assemblies and annotations of Wm82, Lee, Jidou 17, Zh13
  • Many genes to look up among non-reference accessions? Download a correspondence table for all pangene accessions .

If you have extensive programmatic work among arbitrary accessions, see more here:

Download the Glycine "clust"-format pangene file and the pandagma package.
Within that package, the scripts bin/gene_trans_db_load.sh and bin/gene_translate.pl can be used to translate from one annotation to another indicated annotation, for any given list of annotations and desired target annotation; and bin/pangene_tabularize.pl can be used to convert from the pangene "clust.tsv" format to a tabular format, for an indicated list of annotations.
In fact, pangene_tabularize.pl was used to generate the linked pangene table...tsv files linked above; so those may be sufficient for most purposes.

Expression profile, phylogenetic tree, and coexpression network for a single gene.

  • you can view the expression profile,
  • view the phylogenetic tree of orthologous genes with expression values,
  • and view profile neighbors (genes with similar co-expression profiles).
This is available for the Williams 82 assembly 2 annotation (glyma.Wm82.gnm2.ann1.).
